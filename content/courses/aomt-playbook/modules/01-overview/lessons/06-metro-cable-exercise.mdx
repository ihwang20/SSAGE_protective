# Metro Cable Exercise: Playbook Overview

Apply the two core principles — Data First and AI First — to the Metro Cable case before the diagnostic work begins.

<Callout type="concept" title="The Starting Point">
Metro Cable's executive committee has flagged a dispatch accuracy problem. The engagement team has been briefed but has not yet seen any data. The first decisions you make will shape the entire investigation.
</Callout>
